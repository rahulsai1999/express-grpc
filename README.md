# gRPC API

[Postman Docs](https://documenter.getpostman.com/view/5649815/Szf6Yoh7?version=latest)

## Install Dependencies

```shell
git clone https://github.com/rahulsai1999/express-grpc.git
cd express-grpc
npm install
```

## Run gRPC Server

`npm run server`

## Run Express Server

`npm start`
